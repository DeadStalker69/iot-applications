Activity 1: IOT Apps

We needed to seartch about IOT applications, wat they are and their main features. IOT applications are software services or software as a service (SaaS) which display and analyse the data availible on its network, which is collecetd from its own set of sensors. it can strategically analys the data and use the data to execute any sort of process in real time over the internet. IOt applications are integrated with ine another and work together to create a cohesive automated enviornment which is also secure and needes minimal human intervention.

Some examples of IOT applicatios are:

  • Home security
  • Motion detection
  • activity tracker
  • Augmented reality glasses
  • Smart watch
  • Smart water bottle
  • Smart city
  • Traffic management
  • Livestock tracking
